# WeatherNow — Android Weather App

A simple weather app built with Kotlin, MVVM, Retrofit and Material 3.  
**Note:** Add your OpenWeather API key to `local.properties` as `WEATHER_API_KEY=your_key`.

## Setup
1. Clone repo
2. Copy `local.properties.example` → `local.properties` and add your key.
3. Open in Android Studio, Build & Run.

## Features
- Search by city
- Displays temperature, humidity and description
- Material 3 UI + Lottie loading animation
